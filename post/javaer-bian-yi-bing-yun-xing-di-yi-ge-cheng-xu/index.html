Java（二） 编译并运行第一个程序

2022-01-03 1 min read # Java # 编程

如何运行？先编译在运行！
1.编译命令：
在dos命令窗口输入
   javac java源文件路径
或者
   javac "把java源文件直接拖进到DOS命令窗口"
最便捷
   打开计算机-->打开一些文件夹-->在地址栏上直接输入cmd回车，这样直接就过去了。
2.运行命令：
在dos命令窗口输入
   java 类名 //这里必须是在编译的文件的目录下
