Python作业解决

2021-10-23 2 min read # 编程 # Python

要求:

读取Attendance.txt文件内的出勤统计表,存放位置任意.
计算各人的请假天数
计算各人的出勤率
出勤率80%以上为"良好",包括80%
出勤率80%以下为"较差"
计算集体出勤率.

代码：

f=open('C:\\Users\\abc\\Desktop\\python前五章阶段性作业\\Attendance.txt', mode='r')   #打开txt
next(f)#从第二行开始
b=0#总出勤的变量初始
c=0#总天数的变量初始
for l in f:#遍历列表
    #print(l)#打印行
    O=l.count('O')#查O有几个
    X=l.count('X')#X有几个
    a=O/(X+O)*100#计算个人的出勤率
    z=l[:3]#提取列表前3个元素
    print(z,end='')#不换行输出前三个元素
    print('出勤', O)#输出个人出勤天数
    print('缺勤', X)#输出个人缺勤天数
    print('出勤率为：%.0f %%'%a)#输出个人的出勤率
    if a>80:#判断出勤率是否大于80，出勤率80%以上为"良好",包括80%出勤率80%以下为"较差"
        print('良好')#出勤率80%以上为"良好"
    else:
        print('较差')#包括80%出勤率80%以下为"较差"
    b=O+b#计算总出勤
    c=X+O+c#计算总天数
#print(b)
#print(c)
d=b/c*100#计算总出勤率
print('\n    总出勤率为：    %.0f %%'%d)#输出总出勤率
