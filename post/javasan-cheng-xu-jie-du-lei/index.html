Java（三） 程序解读--类

2022-01-03 1 min read # Java # 编程

1、一个java源文件中可以定义多个class。

2、 public的类不是必须的。可以没有。

3、在源文件中只要有一个class的定义，那么必然会对应生成一个class文件。

4、public的类可以没有，但如果有的话，public修饰的类名必须和源文件名保持一致。

5、public的类有也只能有1个。
