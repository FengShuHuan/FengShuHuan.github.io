Java（五） 变量

2022-01-04 2 min read # Java # 编程

1.什么是变量

   变量就是内存中 存储数据的最基本的单元（就是存数据的“盒子”）

2.变量的数据类型有什么用

   不同的数据类型在内存中分配的空间大小不同，JVM是根据不同的数据类型分配不同大小的空间
   例：int分配4个字节空间
         1字节=8比特位（bit）
         1比特位=一个1或者0（二进制）

3.变量的三要素

   数据类型   决定空间大小
   变量名       通过名方便访问
   值             数据

4.命名规则（var是标识符，依照标识符的命名规则和规范）

   由数字，字母，下划线，美元符号组成
   不能以数字开头，不能有空格
   关键字不可作变量名
   变量名不可以重复（大小写是不一样的）
   理论上长度无限制

5.命名规范

   见名知意
   驼峰命名
   首字母小写，后面单词的首字母大写

6.声明与赋值（同C语言）

   声明：   类型     变量名 ；
   （注意：变量名必须先声明并赋值后才能访问，没有赋值就相当于这个空间没有被开辟）
   赋值：  变量名   =    值 ；
